void flashLED ( uint16_t  on_count,
uint16_t  off_count 
)

LEDを指定した時間間隔で点滅させる
例.LEDを100msecの間on, 50msecの間offで点滅させるには
on_count = 100, off_count = 50
LEDを常時点灯する場合は
turnLED(1)
LEDを消灯する場合は
turnLED(0)
